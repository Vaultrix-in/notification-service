# 🔔 Vaultrix Notification Service

Universal event-driven notification hub for the platform.

## 📡 Channels
- **SMTP**: Automated Gmail integration for order confirmations.
- **Webhooks**: Internal event routing.

## 🛠️ Setup
Requires SMTP credentials injected via the Helm `values.yaml` secrets.

- **Trivy**: Container image security scanning.
